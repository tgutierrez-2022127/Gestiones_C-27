 Gestiones C-27

 Descripción
Sistema de gestión de incidentes para el salón C-27.

 Requisitos
- Node.js
- TypeScript

 Instalación
npm install

 Uso
npx ts-node src/index.ts

 Características
- Registro de incidentes
- Prioridades: Alta, Media, Baja
- Reporte ordenado por prioridad
- Patrón DTO

 Estructura del Proyecto
src/
├── types/          Tipos de datos
├── interfaces/     Interfaces
├── dtos/           DTOs
├── services/       Lógica de negocio
├── utils/          Utilidades
└── index.ts        Programa principal
